### DEF-002 · Checkout uses original mock quantities, ignoring cart edits
**Test:** T07-02  
**Severity:** Critical  
**File:** `src/screens/CheckoutScreen.jsx:16, 62–63`  
**Code:**
```jsx
const subtotal = cartItems.reduce((s, c) => s + c.price * c.qty, 0);
// ...

```
**Expected:** Checkout should reflect the quantities the user set in the cart screen (via the `+`/`−` steppers).  
**Actual:** `CheckoutScreen` imports `cartItems` directly from `mockData.js` and reads `c.qty` — the original value. Quantity changes made in `ShoppingCartScreen` are held in local `useState` and are discarded on navigation. The checkout always shows the original mock quantities (1, 2, 1).

### DEF-003 · Quantity changes in cart do not persist to checkout
**Test:** T06-09  
**Severity:** Critical  
**File:** `src/screens/ShoppingCartScreen.jsx:8–14`, `src/screens/CheckoutScreen.jsx:16`  
**Code:**
```jsx
// ShoppingCartScreen — local state only
const [quantities, setQuantities] = useState(
  Object.fromEntries(cartItems.map(c => [c.id, c.qty]))
);
// CheckoutScreen — reads directly from mockData, ignores above
const subtotal = cartItems.reduce((s, c) => s + c.price * c.qty, 0);
```
**Expected:** User-modified quantities flow from Cart → Checkout via shared state (Context, URL params, or global store).  
**Actual:** No shared state mechanism exists. `ShoppingCartScreen` stores quantities in component-local state that is destroyed when the user navigates away. Root cause: no global state management (no Context, Redux, or Zustand).

### DEF-007 · "Top Sales" sort breaks on "k+" and decimal sold strings
**Test:** T03-10  
**Severity:** High  
**File:** `src/screens/SearchResultsScreen.jsx:38`  
**Code:**
```jsx
if (activeSort === 'Top Sales') return parseInt(b.sold) - parseInt(a.sold);
```
**Expected:** Products sort correctly by sold count descending (e.g. "25k+" > "8.9k" > "5.2k" > "3.4k").  
**Actual:** `parseInt` stops at the first non-numeric character: `parseInt("25k+") = 25`, `parseInt("8.9k") = 8`, `parseInt("2.5k") = 2`. This causes products with 2,500 real sales to rank below products with 25 real sales. Correct order is not produced.
